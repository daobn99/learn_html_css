CSS Units

1. Absolute units

  • px
  • pt
  • cm
  • mm
  • inch
  • pc

2. Relative units

  • % (phụ thuộc vào thẻ gần nhất chứa nó)
  • rem (phụ thuộc vào thẻ html)
  • em (phụ thuộc vào thẻ gần nhất (ít dùng hơn rem))
  • vw (1 vw = 1% bề ngang trình duyệt, không quan tâm thẻ chứa nó kích thước bn)
  • vh (1 vh = 1% bề dọc trình duyệt.....)
